## GGFriend APIサーバー
[GGFriend](https://github.com/sdk40010/game-invitation-frontend)(ゲーム仲間を募集できるサービス)用のAPIサーバーです。

## 環境
- PHP 7.x
- Laravel 6.x
- MySQL

## 機能
- 認証API
- 募集API
- コメントAPI
- 返信API
- 参加API
- タグAPI
- ユーザーAPI
- フォローAPI

これらのAPIを利用したGGFriendの機能の詳細は[こちら](https://github.com/sdk40010/game-invitation-frontend#%E6%A9%9F%E8%83%BD)で確認できます。
